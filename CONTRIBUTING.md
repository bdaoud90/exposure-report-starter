# Contributing

Thank you for considering a contribution to the exposure report starter project.
This repository is intentionally minimal and uses neutral, synthetic examples.

## Ways to Contribute
- Propose enhancements to documentation or templates.
- Suggest improvements to structure, naming, or clarity.
- Report gaps in baseline guidance or developer workflows.

## Getting Started
1. Create a new branch from `main`.
2. Make focused changes that align with the repository purpose.
3. Keep example content synthetic and avoid real-world data.

## Commit and Pull Request Guidelines
- Use concise, descriptive commit messages.
- Include context in your pull request summary.
- Link to related issues when applicable.

## Code of Conduct
Please be respectful and collaborative in all interactions.
